Software Requirement Specifications

Project Overview

The AI Instructional Effectiveness Project aims to analyze transcripts of instructional sessions and provide recommendations for improving the effectiveness of the instruction. The project will use natural language processing techniques to identify questions and analyze the effectiveness of the responses given by the instructor. Based on this analysis, the system will provide recommendations for improving the instructional effectiveness.

Functional Requirements

  • The system shall be able to accept transcripts in text format.
  • The system shall be able to identify questions in the transcript.
  • The system shall be able to analyze the effectiveness of the responses given by the instructor.
  • The system shall be able to provide recommendations for improving the instructional effectiveness.

Non-Functional Requirements

  • The system shall have a response time of no more than 5 seconds.
  • The system shall be able to handle transcripts of up to 10,000 words.
  • The system shall be able to handle multiple users simultaneously.
  • The system shall be able to maintain the confidentiality of the transcripts.

Constraints

  • The system shall be developed using Python programming language.
  • The system shall use natural language processing techniques for identifying questions and analyzing responses.
  • The system shall be deployed on a cloud-based platform.

Assumptions

  • The transcripts will be in English language only.
  • The transcripts will be in text format.
  • The transcripts will be provided by the user.
